Abaixo mostro alguns projetos que desenvolvi em âmbito pessoal e educacional para aprendizado, possui detalhes de cada projeto assim como as tecnologias que foram utilizadas.

Rocketpay

O Rocketpay é um componente que simula o formulário de preenchimento de cartão de crédito, onde é possível adicionar máscara aos inputs e atualizar elementos HTML via DOM.

Esse projeto foi desenvolvido com as seguintes tecnologias:

  • HTML
  • CSS
  • JavaScript
  • JSON
  • NodeJs
  • Npm
  • Vite
Sistema verificador máscara

Projeto criado para apresentação de trabalho da máteria de IA com foco em redes neurais, consiste em um verificador de uso de máscara com três classificações sendo elas: Rosto sem máscara, rosto com máscara e uso errado de máscara.

Esse projeto foi desenvolvido com as seguintes tecnologias:

  • HTML
  • JavaScript
  • Bootstrap
  • TensorFlow
Pokedex

Projeto desenvolvido com video aula do canal DevemDobro, com o intuito de adquirir conhecimentos em JavaScript. O projeto trabalha a manipulação de elementos

Esse projeto foi desenvolvido com as seguintes tecnologias:

  • HTML
  • CSS
  • JavaScript
Dark-mode / Light-mode

Esse projeto foi desenvolvido no bootcamp da Digital Innovation One no módulo de JavaScript, consiste em uma tela que se inicia em Light-Mode e possui um botão de Dark-Mode, o botão ao ser clicado troca o tema.
